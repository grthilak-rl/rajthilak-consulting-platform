This project is a production-grade personal consulting platform designed to:

- Showcase my technical portfolio and real-world projects
- Allow companies and individuals to submit project or job requirements
- Provide a private dashboard to manage incoming requests, track progress, and deliver outcomes

## Features (MVP)

Public:
- Portfolio & services showcase
- Requirement submission form
- Contact form

Admin:
- Secure login
- Requirement management dashboard
- Status & progress tracking
- Notes per requirement

## Tech Stack

Frontend:
- React + Vite
- Tailwind CSS

Backend:
- FastAPI
- PostgreSQL / SQLite
- JWT Authentication

Infra:
- Docker + Docker Compose
- Nginx (optional for prod)

## Why this exists

This platform is intentionally built as a real system instead of a static portfolio website to demonstrate:
- System design
- API design
- Authentication & authorization
- Full-stack integration
- Platform thinking
